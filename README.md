# Stock Market Virtual App

A full-stack web application that simulates a stock market environment where users can buy and sell stocks with virtual money. The app updates stock prices live based on user transactions.

---

## Technologies Used

Frontend: React.js, HTML, CSS, JavaScript  
Backend: Node.js, Express.js, MongoDB  
Others: Socket.io for live updates, JWT Authentication (if implemented)

---

## Features

- User registration and login
- Buy and sell virtual stocks
- Live stock price updates based on user activity
- Portfolio overview
- Realistic virtual trading simulation

---

## How to Run Locally

1. Backend:

````bash
cd backend
npm install
npm start

2.Frontend
```bash
cd frontend
npm install
npm start
````
